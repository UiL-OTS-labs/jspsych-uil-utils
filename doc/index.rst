jspsych-uil-utils documentation
===============================

This is the documentation belonging to the **jspsych-uil-utils**. A set of utilies, that
helps people from the **IlsLabs** to work with jsPsych in an online environment. The
work on the jspsych-uil-utils has started during the Covid pandemic. At the time of
writing the IlsLabs were called the UiL-OTS labs (or uil in short), hence the name.

In the following you can see the important parts of the utils (jspsych-uil-utils).
There is a quickstart in order to setup your experiment.
